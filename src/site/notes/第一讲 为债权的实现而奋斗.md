## 1 民法对债权的保障
1.  民法上的担保制度，是指担保债权实现的法律制度。
2. 债权的实现之所以成为问题，是因为相对于物权，债权的效力较弱。债权作为对人权和请求权，具有相对性，其实现有赖于债务人的给付行为，因此债权之实现，须取决于债务人的清偿能力。
3. 从法律的角度来说，保障债权得以实现，是一个国家法治文明程度的重要标志之一。
4. 一般认为，现代民法对债权的保障大致可以分为三个层次：一是强化债权的效力；二是健全担保制度；三是发展合同上的义务群。
	1. 债权的效力是债权保障的内在动力。
		1. 法律为保障债权的存续与实现，赋予债权如下效力：请求力、执行力、依法自力实现、处分权能和保持力。
		2. 从民法的发展情况看，债权的效力呈不断扩张之势。
	2. 健全担保制度。
		3. 债权本身的效力虽然能够在一定程度上保障债权的实现，但在债务人的财产不足以清偿全部债务的情形下，债权仍有不能实现的风险。这个时候，担保制度的重要性就凸显出来。
		4. 担保分为典型担保和非典型担保；
			1. 典型担保，包括人的担保（保证）、物的担保（抵押权、质权、留置权）和金钱担保（定金）；
			2. 非典型担保，主要是让与担保、所有权保留、融资租赁以及有追索权的保理等；
	3. 民事责任制度在保障债权的现实方面也发挥着重要的作用，尤其是在民事责任制度中的违约责任制度。
		1. 违约责任的追究依合同当事人违反合同义务为前提，因此判断合同当事人是否违反合同义务，就成为能否利用违约责任制度保障债权实现的先决条件。而要判断合同当事人是否违反合同义务，就须先知道合同包含那些义务。
		2. 合同关系上义务群的发展，在 20 世纪的合同法中，是一道靓丽的风景线，甚至有学者称，“债法关系上的义务群，乃债法的核心问题”，现代债法的发展，在某种意义上，可以说是债之关系上义务群的发展。
## 2 债的保全与债的担保
1. 当债务人的责任财产应该增加而没有增加，从而影响债权的实现时，债权人就可以代债务人的位向此债务人主张权利，从而使得债务人的责任财产能够扩大，进而提高债务人的清偿能力。
2. 域外多数国家或地区关于债权人代位权的规定，债权人代债务人的位向此债务人主张权利，其法律后果应归属于债务人，然后再由债务人向债权人清偿债务，这个规则叫“入库规则”。但入库规则不利于鼓励或激励债权人代为债务人主张权利，所以我们法律没有按照入库规则设计债权人代位权，而是明确规定债权人代债务人向次债务人主张权利时，其后果直接归属于债权人，而不是归属于债务人。
3. 债权人的撤销权所要解决的问题是，当债务人的责任财产“人为”地减少，从而影响到债务人的清偿能力时，债权人可以请求人民法院撤销债务人无偿赠送的行为或者是以明显低价处分其财产的行为。
4. 债的保全制度有一个非常重要的特征，关注的都是债务人自身的责任财产。债务人的责任财产该增加的没有增加，不该减少的减少了，从而影响到债权的实现，这个时候债权人就可以动用债的保全制度来维护其利益。所谓债的“保全”，保全的是债务人的责任财产，就是防止债务人的责任财产“人为”地减少。
5. 债的保全制度，只能解决债务人“人为”减少责任财产给债权实现带来的风险，无法仿制债务人正常经营过程中产生的风险。也正是在这个问题上，担保制度发挥着非常重要的作用，因为担保制度不仅可以将第三人的财产纳入责任财产，而且在设定担保物权的情况下，还可以使享有担保物权的值啊权人优先于其他债权人受偿。
## 3 担保制度的运行机理
1. 保证制度和债的保全制度有一个明星的区别：债的保全关注的是债务人自身的责任财产，即该增加的没有增加，不该减少的减少了，而保证实际上是把保证人的责任财产纳入整个责任财产，其本质是扩大责任财产的范围。
2. 保证制度虽然扩大了责任财产的范围，但保证人的清偿能力也会面临上面同样的问题，也是处于变动状态，因此保证制度只能降低风险，并不能彻底避免风险。
3. 担保物权实际上有一个重要的特征，就是以债务人或者第三人提供的特定财产来保障债权优先得到实现，从而避免责任财产变化带来的风险。
4. 定金的担保功能就是通过定金罚则来实现的。定金具有双方担保的功能，同时担保双方履行合同：不仅担保交付定金的一方当事人履行合同，也担保接受定金的一方当事人履行合同。这是定金的一个很重要的特点。
